ğŸ’¡ WHY THIS WORKS

Vercel Environment Variables:
  - Can be set in Vercel UI (Dashboard)
  - Can be set in vercel.json (but NOT as references)
  - Can be referenced in GitHub Actions with @SECRET_NAME syntax
  - Our backend code reads them from process.env automatically

The @ syntax only works in specific Vercel services, not for regular env vars.
So we removed it and use the UI instead (simpler, more reliable).
